# 云计算
```md
“云计算”概念由Google提出，一如其名，这是一个美丽的网络应用模式
引用伯克利大学云计算白皮书里面的定义：
	云计算包含互联网上的应用服务及在数据中心提供这些服务的软硬件设施
```
```md
云计算是是分布式处理(Distributed Computing)、
	并行处理(Parallel Computing)和网格计算(Grid Computing)的发展
是透过网络将庞大的计算处理程序自动分拆成无数个较小的子程序
	再交由多部服务器所组成的庞大系统经计算分析之后将处理结果回传给用户。
```
```md
通过云计算技术
	网络服务提供者可以在数秒之内，处理数以千万计甚至亿计的信息
		达到和“超级计算机”同样强大的网络服务
```
```md
云计算最初的目标是对资源的管理
	管理的主要是计算资源，网络资源，存储资源三个方面
```
